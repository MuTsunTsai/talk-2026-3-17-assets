# LogiTrack

電商物流後台管理系統的前端範例專案，以展示介面設計為主，並沒有實際串接後端 API，所有資料均為本地假資料。

## 頁面說明

| 頁面 | 檔案 | 內容 |
|---|---|---|
| 總覽儀表板 | `index.html` | KPI 統計卡片、訂單趨勢折線圖、最新訂單摘要 |
| 訂單管理 | `orders.html` | 訂單列表、篩選、操作選單 |
| 出貨追蹤 | `shipments.html` | 物流軌跡時間軸 |
| 商品庫存 | `products.html` | 商品列表、庫存快速調整 |
| 系統設定 | `settings.html` | 帳號資訊、通知偏好設定 |

## 技術

- Bootstrap 5（CDN）
- Vanilla JavaScript
- 無需任何建置工具或套件安裝

## 執行方式

### 直接開啟

用瀏覽器直接開啟 `index.html` 即可瀏覽。
